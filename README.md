# Trinity Process Solutions

Monorepo for the Trinity Process Solutions website — a boutique IT
consultancy site with a dark-luxury aesthetic (near-black + antique gold).

## Structure

```
/frontend   Next.js (App Router, TypeScript, Tailwind) — the public website
/backend    Node/Express (TypeScript) — API for the website
```

See `Trinity_Process_Solutions_Brand_Guidelines.md` for the full brand spec
(colors, typography, voice) and `CLAUDE.md` for project conventions.

## Development

From the repo root:

```
npm install
npm run dev
```

This runs the frontend and backend dev servers concurrently.

### Frontend

See `/frontend` for the Next.js app.

### Backend

See `/backend` for the Express API.
